In this project I learned how to display images in different projections

Descibe how you displayed the map in different projections using QGIS:

To display the map in different projections using QGIS, I first imported my vector layer of the world map. Then, I installed the Indicatrix plugin and used it to add a layer of indicator circles that show how each projection distorts the map. Finally, to change the projection, I changed the ESRI in the bottom right to the corresponding map projection that I wanted and applied it to my map.
